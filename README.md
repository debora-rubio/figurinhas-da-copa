# Sistema de Figurinhas da Copa

Projeto desenvolvido para a disciplina de Estrutura de Dados da Fatec Rio Claro.

## Objetivo

Desenvolver um sistema para gerenciamento de figurinhas da Copa do Mundo, permitindo armazenar, consultar, trocar e organizar figurinhas utilizando estruturas de dados implementadas manualmente.

## Estruturas de Dados Utilizadas

- Lista Encadeada
- Fila FIFO
- Nós Encadeados


## Funcionalidades

- Inserir figurinha
- Remover figurinha
- Consultar figurinha
- Ver álbum completo
- Ver porcentagem concluída
- Armazenar figurinhas repetidas
- Mostrar figurinhas repetidas
- Contar figurinhas repetidas
- Buscar por jogador
- Buscar por país/seleção
- Registrar proposta de troca
- Efetuar troca automática
- Salvar dados em arquivo TXT
- Carregar dados de arquivo TXT

## Como Executar

1. Abra o projeto no VS Code.
2. Abra o terminal na pasta do projeto.
3. Execute o comando:

```bash
python main.py
```

4. Utilize o menu exibido no terminal.

## Requisitos

- Python 3.x instalado.
- Não utiliza bibliotecas externas.
